# Print method for `bgms` objects

Minimal console output for `bgms` fit objects.

## Usage

``` r
# S3 method for class 'bgms'
print(x, ...)
```

## Arguments

- x:

  An object of class `bgms`.

- ...:

  Ignored.

## Value

Invisibly returns `x`.
